Cachipun

Instrucciones

  1. Debes ingresar un número de partidas a jugar (solo números positivos)
  2. Debes ingresar una opción:
    • 0 para Papel
    • 1 para Tijeras
    • 2 para Piedra
  3. Por cada partida se te mostrará el resultado (en la consola se muestra el histórico de cada partida)
  4. Terminadas todas las partidas, el sistema te mostrará si ganaste, empataste o perdiste (en la consola te mostrará un resumen de partidas ganadas, empatadas y perdidas.
